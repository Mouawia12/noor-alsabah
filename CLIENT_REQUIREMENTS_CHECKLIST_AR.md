## 1) الافتراضات والأسئلة المفتوحة (تحتاج تأكيد العميل قبل التصميم)

- [ ] **حدود الفاتورة داخل ملف الـ PDF الكبير:** هل كل فاتورة في **صفحة واحدة**؟ أم قد تمتد لعدة صفحات؟ (يحدد طريقة التقسيم — الافتراض المبدئي: فاتورة = صفحة).
- [ ] **نوع الفواتير:** ممسوحة ضوئياً (صور) أم PDF نصي؟ (يحدد هل نحتاج تحويل الصفحات لصور قبل القراءة).
- [ ] **لغة الفواتير/العقود:** عربية، إنجليزية، أم مختلطة؟
- [ ] **توحيد عقود الإيجار:** العميل أرسل نموذجين موحّدين (`20890030945.pdf`, `1719133229.pdf`) — هل كل العقود بنفس القالب؟ (يسهّل الاستخراج كثيراً).
- [ ] **حد أعلى للحجم/العدد** في الرفعة الواحدة (50؟ 100؟ أكثر؟) — لضبط أداء المعالجة الخلفية.
- [ ] **قنوات التنبيه:** داخل النظام فقط، أم بريد/واتساب/SMS أيضاً؟ (المحادثة ذكرت "النظام الداخلي").
- [ ] **حد الثقة (confidence)** الذي يُمرَّر تلقائياً مقابل الذي يُحوَّل للمراجعة اليدوية.
- [ ] **التكلفة التشغيلية لاستدعاءات الـ AI** على من؟ (لكل 50–100 فاتورة تكلفة API — يُتفق عليها).

## 2) القرارات المحسومة + Stack

> **قرارات مثبّتة مع العميل/المنفّذ:**
> 1. **محرك الاستخراج:** OpenAI (رؤية) — مع طبقة خدمات معزولة قابلة لتبديل المحرك.
> 2. **حدود الفاتورة:** كشف ذكي بالـAI (يدعم فاتورة-بصفحة / متعددة الصفحات / عدة فواتير بصفحة).
> 3. **اللغات:** مختلط عربي/إنجليزي.
> 4. **قنوات التنبيه:** داخل النظام + البريد الإلكتروني.

| الحاجة | الأداة |
|--------|--------|
| استخراج بيانات الفواتير/العقود (رؤية + JSON منظّم) | **OpenAI Vision** — `gpt-4o-mini` للحجم الكبير اقتصادياً + تصعيد `gpt-4o` عند انخفاض الثقة، عبر **Structured Outputs (json_schema)** لضمان مخرجات منظّمة موثوقة |
| كشف حدود الفواتير | تصنيف "بداية فاتورة جديدة؟" بالـAI + heuristics (تغيّر الرقم/المورد) |
| تقسيم/استيراد صفحات PDF | `setasign/fpdi` (+ tcpdf الموجود) |
| تحويل صفحة PDF ممسوحة إلى صورة | Imagick + Ghostscript (تبعية سيرفر) |
| قراءة نص PDF النصي | `smalot/pdfparser` |
| المعالجة الخلفية للدفعات الكبيرة | Laravel Queue (`database` أو Redis) + Jobs |
| عميل HTTP | `guzzlehttp/guzzle` (موجود) |
| التنبيهات | إشعارات Laravel (قناة database داخل النظام + قناة mail) |
| تصدير التقارير | dompdf + phpspreadsheet (موجودان) |
